Compliation:
To compile the program run the Makefile.

Testing:
Test cases can be found in the TestCases Folder
To run the tests use the following:
./scrabble < ./testCases/[TestName]/[TestName].input

./scrabble <testCases/CompleteGamePlay/CompleteGamePlay.input >testCases/CompleteGamePlay/CompleteGamePlay.gameoutput
